# USBGuard Workshop Lab - RHEL 10

## Workshop Overview

This hands-on lab introduces USBGuard, a software framework for implementing USB device authorization policies on Linux systems. Learn to protect your RHEL 10 systems from unauthorized USB devices and potential security threats.

## Learning Objectives

By the end of this workshop, you will be able to:
- Understand USBGuard architecture and use cases
- Install and configure USBGuard on RHEL 10
- Create and manage USB device authorization policies
- Use USBGuard CLI tools to control device access
- Implement security policies for enterprise environments
- Troubleshoot common USBGuard issues

## Prerequisites

- Red Hat Enterprise Linux 10 system (physical or virtual)
- Root or sudo access
- Basic Linux command-line knowledge
- At least one USB device for testing

## Lab Environment

- **OS**: Red Hat Enterprise Linux 10
- **Required Packages**: usbguard, usbguard-tools
- **Time Required**: 10-15 minutes

## Workshop Structure
