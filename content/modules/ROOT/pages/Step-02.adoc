== Installing the USBGuard packages

## Step 2

### 2.1 Check System Information

First, verify you're running RHEL 10:

[source,ini,role=execute,subs=attributes+]
----
cat /etc/redhat-release
uname -r
----

Expected output should show RHEL 10.

### 2.2 Install USBGuard

Install USBGuard and related tools:

[source,ini,role=execute,subs=attributes+]
----
sudo dnf install -y usbguard usbguard-tools
----

### 2.3 Verify Installation

Check the installed version:

[source,ini,role=execute,subs=attributes+]
----
rpm -qa | grep usbguard
usbguard --version
----

To Move to the next step, click the "Next" button.
