== Installing the USBGuard packages

## Step 2

### 2.1 Check System Information

First, verify you're running RHEL 10:

[source,ini,role=execute,subs=attributes+]
----
cat /etc/redhat-release
uname -r
----

Expected output should show RHEL 10.

### 2.2 Install USBGuard

Install USBGuard and related tools:

[source,ini,role=execute,subs=attributes+]
----
sudo dnf install -y usbguard usbguard-tools
----

### 2.3 Verify Installation

Check the installed version:

[source,ini,role=execute,subs=attributes+]
----
rpm -qa | grep usbguard
usbguard --version
----

### 2.4 Understand USBGuard Components

USBGuard consists of several key components:

- **usbguard-daemon**: The main service that enforces USB policies
- **usbguard**: CLI tool for managing the daemon
- **usbguard-applet-qt**: GUI application (optional)
- **Configuration Files**:
  - `/etc/usbguard/usbguard-daemon.conf` - Daemon configuration
  - `/etc/usbguard/rules.conf` - Policy rules

### 2.5 Check Current USB Devices

Before starting USBGuard, list currently connected USB devices:

[source,ini,role=execute,subs=attributes+]
----
lsusb
----
