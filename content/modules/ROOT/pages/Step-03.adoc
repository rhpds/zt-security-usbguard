## Step 3 Configure USBGuard

### 3.1 Generate Initial Policy

USBGuard can generate an initial policy based on currently connected devices:

[source,bash,role="execute",subs=attributes+]
----
sudo usbguard generate-policy > /tmp/initial-rules.conf
cat /tmp/initial-rules.conf
----

**IMPORTANT**: Review this file carefully. It shows rules for all currently connected USB devices.

### 3.2 Install the Initial Policy

Copy the generated policy to the proper location:

[source,bash,role="execute",subs=attributes+]
----
sudo install -m 0600 -o root -g root /tmp/initial-rules.conf /etc/usbguard/rules.conf
----

Verify the file permissions:

[source,bash,role="execute",subs=attributes+]
----
ls -l /etc/usbguard/rules.conf
----

Should show: `-rw------- 1 root root`

### 3.3 Start USBGuard Service

Enable the USBGuard daemon:

[source,bash,role="execute",subs=attributes+]
----
sudo systemctl enable usbguard.service
----

Start the USBGuard daemon:

[source,bash,role="execute",subs=attributes+]
----
sudo systemctl start usbguard.service
----

### 3.4 Verify Service Status

Check that the service is running:

[source,bash,role="execute",subs=attributes+]
----
sudo systemctl status usbguard.service
----

Look for "active (running)" status.

You might have to close out of the terminal with "Ctrl+c".

### 3.5 Test Basic Functionality

List currently authorized devices:

[source,bash,role="execute",subs=attributes+]
----
sudo usbguard list-devices
----

Now that we have it installed and configured, lets cover how to veriy it in the next section.
