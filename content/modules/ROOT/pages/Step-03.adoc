Configure USBGuard


Step 3.1 - Create an initial rule set:
[source,ini,role=execute,subs=attributes+]
----
usbguard generate-policy > /etc/usbguard/rules.conf
----


Step 3.2 - Start The Service:
Start the usbguard daemon and ensure that it starts automatically on boot
[source,ini,role=execute,subs=attributes+]
----
systemctl enable --now usbguard
----

Now that we have it installed and configured, lets cover how to veriy it in the next section.
